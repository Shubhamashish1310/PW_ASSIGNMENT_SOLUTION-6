When you target the same elements with inline, internal, and external CSS, the following happens:

  • The inline CSS will have the highest priority and will override the internal and external CSS.
  • The internal CSS will have the second highest priority and will override the external CSS, but not the inline CSS.
  • The external CSS will have the lowest priority and will be overridden by the internal and inline CSS.
